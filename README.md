# MassSpec_ExampleWorkflow
Example Workflow with example data for processing and analysing DIA proteomics data. This workflow starts with DIA-NN .parquet search engine output, performs basic pre-processing, PCA clustering analysis and limma differential expression analysis.

There are two skript options:
1. DataAnalysis_Slim: regular differential expression analysis
2. DAtaAnalysis_Slim_pairedDE: for paired testing setups
